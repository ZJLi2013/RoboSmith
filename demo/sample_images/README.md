# Sample Images for Hunyuan3D-2.1 Demo

Place reference images here for image-to-3D generation.

If no image is provided, the demo script will automatically look for
`assets/demo.png` in the Hunyuan3D-2.1 repo directory.

Recommended:
- PNG or JPEG, single object on clean background
- 512x512 or higher resolution
- Well-lit, minimal shadows

Example filenames:
- `mug.png` — a ceramic mug
- `bowl.png` — a white bowl
- `bottle.png` — a glass bottle
